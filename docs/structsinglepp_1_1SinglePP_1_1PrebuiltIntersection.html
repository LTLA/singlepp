Markers singlepp::SinglePP::PrebuiltIntersection::markers

A vector of vectors of ranked marker genes to be used in the classification. Values are indices into the mat_subset and ref_subset vectors for the respective matrices. The set of marker genes is typically a subset of those in the input markers in build().
